# Forest Field Studio

This folder contains an alternate Streamlit experience for the forest calculation workflow.

It keeps the original calculation engine from:

- `../run_forest_calculation.py`
- `../template.xlsx`
- `../species_reference_master_v1.xlsx`
- `../forest_component_7.xlsx`

while presenting the workflow in a more polished, Forest-inspired interface.

Run it from the repository root with:

```bash
streamlit run forest_landing_app/app.py
```

The original `app.py` remains unchanged and can still be launched separately.
